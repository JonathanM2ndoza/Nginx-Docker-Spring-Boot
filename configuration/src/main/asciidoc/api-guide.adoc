[[overview-http-headers]]
== HTTP Headers
Los endpoints deben tener los siguientes elementos en el header del REQUEST:

|===
|Header|Type|Structure|Description

|Json Web Token
|Security
|`+Authorization: {token}+`
|Token de acceso JWT

|===
* Enviar estos headers en los request, a menos que se indique lo contrario.
